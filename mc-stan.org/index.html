Stan

Stan is a probabilistic programming language implementing inference with

  • Markov chain Monte Carlo sampling (NUTS, HMC)
  • Optimization-based point estimation (BFGS)

Stan is implemented in C++, with interfaces for R, Python, and the command line.

Stan is freedom-respecting, open-source software.
